# EvalGenAI 🔥

## Overview
AI-powered academic assistant that:
- Answers questions using RAG
- Predicts important exam questions

## Features
- Dual RAG architecture
- Diagram-aware answers
- Question prediction engine
- Agent-based routing

## Architecture
User → Agent → RAG (Answer / Prediction)

## Tech Stack
- Python
- ChromaDB
- Sentence Transformers
- Gemini API

## Run

pip install -r requirements.txt  
python main.py
